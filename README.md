📌 Overview

This project predicts the risk of lung cancer using Machine Learning.
Users can enter medical parameters through a simple web interface and get instant predictions.

---

🚀 Features

- User-friendly web interface (Streamlit)
- Real-time prediction
- Fast and lightweight ML model
- Easy to use for non-technical users

---

🛠️ Tech Stack

- Python
- Scikit-learn
- Streamlit
- Joblib

⚙️ How to Run Locally

1. Clone the repository:

git clone https://github.com/your-username/Lung_Cancer_Detection.git

2. Navigate to folder:

cd Lung_Cancer_Detection

3. Install dependencies:

pip install -r requirements.txt

4. Run the app:

streamlit run app.py
